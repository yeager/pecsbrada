# PECS-bräda

Digital PECS communication board with ARASAAC pictograms and text-to-speech.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install pecsbrada
```

### Fedora/RPM
```bash
sudo dnf install pecsbrada
```
